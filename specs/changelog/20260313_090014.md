# Changelog — 20260313_090014

## Update
- Mounted `POST /api/v1/boards/:id/background` and `DELETE /api/v1/boards/:id/background` routes in `server/extensions/board/api/index.ts`.

## New
- `server/extensions/board/api/uploadBackground.ts` — Accepts `multipart/form-data` with a `background` field (JPEG or PNG, ≤ 10 MB). Uploads to S3 at `board-backgrounds/{boardId}/background.{ext}`, updates `boards.background`, and emits `board.background_changed` WS event. Requires ADMIN role.
- `server/extensions/board/api/deleteBackground.ts` — Clears `boards.background` to `null`, deletes the S3 object, and emits `board.background_changed` WS event. Requires ADMIN role. Idempotent (no error if no background was set).
- `specs/tests/board-background-upload.md` — Playwright MCP test scenarios for upload (valid JPEG, invalid MIME, non-admin 403), delete (returns null, idempotent), and WS event verification.

## Technical Debt
- `extractS3KeyFromUrl` is duplicated between `uploadBackground.ts` and `deleteBackground.ts`. Should be extracted to a shared utility (e.g. `server/common/s3/extractKeyFromUrl.ts`) when a third caller emerges.
