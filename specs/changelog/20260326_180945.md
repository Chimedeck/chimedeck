# Changelog — 2026-03-26 18:09

## Update

**Sprint 120 Part B — Text & Contrast Full Audit**

Completed a comprehensive audit and migration of all remaining hardcoded Tailwind color classes across 39 source files. All text, background, and border colors now use semantic tokens that adapt correctly to both light and dark themes.

### Semantic token replacements applied

| Old class | New class |
|-----------|-----------|
| `bg-white` | `bg-bg-surface` |
| `bg-gray-50`, `bg-gray-100` | `bg-bg-overlay` |
| `bg-gray-200`, `bg-gray-300` | `bg-bg-sunken` |
| `hover:bg-gray-50`, `hover:bg-gray-100` | `hover:bg-bg-overlay` |
| `hover:bg-gray-200` | `hover:bg-bg-sunken` |
| `border-gray-200`, `border-slate-200` | `border-border` |
| `border-gray-300` | `border-border-strong` |
| `divide-slate-100` | `divide-border` |
| `text-gray-400` | `text-subtle` |
| `text-gray-500`, `text-gray-600` | `text-muted` |
| `text-gray-700–900` | `text-base` |
| `text-red-500`, `text-red-600` | `text-danger` |
| `text-green-500`, `text-green-600` | `text-success` |
| `hover:text-black dark:hover:text-white` | `hover:text-base` |
| `bg-gray-900` (loading screen) | `bg-bg-base` |
| `text-blue-500` (spinner) | `text-primary` |

| `bg-red-500 text-white` (notification badge) | `bg-danger text-inverse` |

### Stale `dark:` variants removed

All paired `dark:bg-slate-*` / `dark:bg-gray-*` / `dark:border-slate-*` variants that duplicated what the semantic token already provides were removed. Only semantic token classes remain.
