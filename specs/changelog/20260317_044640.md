# Changelog — 20260317_044640

## Update
- `src/extensions/Card/components/CardDescriptionTiptap.tsx` — Refactored rich-text toolbar from a wrapping multi-button row to a single-line layout. Primary formatting controls (Bold, Italic, Strikethrough, Bullet list) are always visible as icon-only buttons with `aria-label`/`title`. Secondary controls (Numbered list, Quote, Code block) are hidden behind a **+** overflow button at the far right; clicking it opens a dropdown menu. Click-outside detection closes the menu automatically.
