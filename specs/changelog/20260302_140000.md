# Changelog — 20260302_140000

## Update
- `server/index.ts`: Mounted `cardRouter` under `/api/v1/lists/:id/cards` and `/api/v1/cards/:id`.

## New
- `db/migrations/0006_card.ts`: Migration adding the `cards` table (`id`, `list_id`, `title`, `description`, `position`, `archived`, `due_date`, `created_at`, `updated_at`).
- `server/extensions/card/middlewares/requireCardWritable.ts`: Middleware that loads card + board, returns 403 if board is ARCHIVED or card is archived; attaches `card` and `board` to request.
- `server/extensions/card/api/create.ts`: `POST /api/v1/lists/:id/cards` — create card with fractional position (append); validates title ≤ 512 chars; min role MEMBER.
- `server/extensions/card/api/list.ts`: `GET /api/v1/lists/:id/cards` — list active cards sorted by position; min role VIEWER.
- `server/extensions/card/api/get.ts`: `GET /api/v1/cards/:id` — full card detail with empty includes stubs (labels, members, checklist etc. added in sprint 08); min role VIEWER.
- `server/extensions/card/api/update.ts`: `PATCH /api/v1/cards/:id` — update title, description, due_date; blocks on archived cards; min role MEMBER.
- `server/extensions/card/api/archive.ts`: `PATCH /api/v1/cards/:id/archive` — toggle archived state; min role MEMBER.
- `server/extensions/card/api/move.ts`: `POST /api/v1/cards/:id/move` — move card to a target list with fractional position recalculation; validates same-board constraint; min role MEMBER.
- `server/extensions/card/api/duplicate.ts`: `POST /api/v1/cards/:id/duplicate` — copy all scalar fields with new id and position immediately after source; min role MEMBER.
- `server/extensions/card/api/delete.ts`: `DELETE /api/v1/cards/:id` — hard-delete; min role ADMIN.
- `server/extensions/card/api/index.ts`: Card router wiring all eight endpoints.
- `src/extensions/Card/api.ts`: Client API module for all card operations (list, create, get, update, archive, move, duplicate, delete).
- `src/extensions/Card/translations/en.json`: English copy for all card UI strings.
- `src/extensions/Card/components/CardTile.tsx`: Compact card tile for rendering inside `ListColumn`; keyboard accessible.
- `src/extensions/Card/components/CreateCardForm.tsx`: Inline form to create a new card at the bottom of a list.
- `src/extensions/Card/components/CardModal.tsx`: Full-detail overlay modal with title editor, description textarea, archive/move/duplicate sidebar actions.
- `src/extensions/Card/README.md`: Feature documentation covering data model, API routes, move logic, and error responses.
