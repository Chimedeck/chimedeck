## Update
- Updated card update, comment create, and comment update endpoints to extract, resolve, and sync @mentions, wrapping DB writes in a transaction and calling `syncMentions`.
- Board API router extended to serve the new member suggestions endpoint.
- `CardDescription`, `CommentEditor`, `CommentThread`, `CommentItem`, `CardModal`, and its container updated to accept and thread `boardId` for mention autocomplete in both new and edit-inline flows.

## New
- `db/migrations/0016_mentions.ts`: Migration for `mentions` table with unique constraint and indexes.
- `server/common/mentions/parse.ts`: Utility to extract @nickname patterns from text.
- `server/common/mentions/resolve.ts`: Utility to resolve nicknames to user IDs within board membership.
- `server/common/mentions/sync.ts`: Utility to diff and sync mentions in a transaction.
- `server/extensions/board/api/members/suggestions.ts`: Endpoint for mention autocomplete suggestions.
- `src/common/components/MentionInput/`: New components: `MentionInput.tsx`, `MentionChip.tsx`, `MentionSuggestions.tsx`, `useMentionInput.ts`, `renderMentions.tsx` for mention input, chips, suggestions, and rendering.
