## New
- `server/extensions/payment/config/stripe.ts`: Stripe SDK singleton — returns a configured `Stripe` instance if `STRIPE_SECRET_KEY` is set, otherwise `null`.
- `server/extensions/payment/api/index.ts`: `paymentRouter` — handles `POST /api/v1/payments/stripe/create-payment-intent`. Requires auth, validates `amount > 0`, creates a PaymentIntent and returns `{ data: { clientSecret } }`. Returns 503 `stripe-not-configured` if key absent, 400 `invalid-amount` if amount ≤ 0, 502 `stripe-error` on Stripe SDK failure.
- `stripe@20.4.0` installed as a runtime dependency.
