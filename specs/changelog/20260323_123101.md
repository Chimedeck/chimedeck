## New
- `db/migrations/0093_api_tokens.ts`: Migration creating the `api_tokens` table with columns `id`, `user_id` (FK→users CASCADE), `name`, `token_hash` (unique), `token_prefix` (10 chars), `expires_at`, `last_used_at`, `revoked_at`, `created_at`; indexes on `user_id` and `token_hash`.
- `server/extensions/apiToken/mods/generate.ts`: `generateApiToken()` async helper that produces `{ raw, hash, prefix }` — raw is `hf_` + 64 lowercase hex chars (32 random bytes via `crypto.getRandomValues`), hash is `SHA-256(raw)` as hex via `crypto.subtle.digest`, prefix is the first 10 chars of raw for display.
