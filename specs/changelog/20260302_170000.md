## Technical Debt

- `marked` renders user-owned Markdown without HTML sanitisation. For production, add `DOMPurify` or equivalent before the `dangerouslySetInnerHTML` call in `CardDescription`.
- `allLabelsRef` and `boardMembersRef` in the container are local refs; a future iteration should load them into Redux state so multiple consumers can share them without duplicate fetches.
- Checklist rename does not have an optimistic path — it fires the API directly and re-fetches on error. Add an optimistic rename action if rename latency becomes noticeable.
