## Update
- Added `label`, `category`, and `configSchema` fields to `ActionHandler` interface in `common/types.ts` for richer action metadata and validation.
- Updated `engine/registry.ts` to export `getAllActionHandlers()` for action-types API support.
- Modified `automation/api/index.ts` to import actions index, ensuring all action handlers are registered at startup.

## New
- Implemented 14 card action handlers in `engine/actions/card/`: `moveToList`, `moveToTop`, `moveToBottom`, `addLabel`, `removeLabel`, `addMember`, `removeMember`, `setDueDate`, `removeDueDate`, `markDueComplete`, `addComment`, `archive`, `addChecklist`, `mentionMembers`.
- Created `engine/actions/variables.ts` for variable substitution in action configs (supports `{cardName}`, `{boardName}`, `{listName}`, `{date}`, `{dueDate}`, `{triggerMember}`).
- Added `engine/actions/index.ts` to register all card action handlers in the action registry.
