# Changelog — 20260313_093444

## Update
- `src/extensions/Search/hooks/useSearch.ts` — Added optional `type` param (`'board' | 'card'`) to `UseSearchOptions`. The param is forwarded to `searchWorkspace` and included in the `useEffect` dependency array so the search re-runs when the scope changes.
- `src/extensions/Search/components/SearchResultItem.tsx` — Added optional `buttonRef` and `onKeyDown` props to support keyboard focus management from CommandPalette. Added dark-mode Tailwind variants to hover/focus states and text colours.
- `src/layout/AppShell.tsx` — Replaced `SearchModal` import with `CommandPalette`. Updated JSX to use `<CommandPalette>` and updated the comment from "search modal" to "command palette".

## New
- `src/common/components/CommandPalette.tsx` — New command-palette component built on top of `useSearch` and `SearchResultItem`. Features:
  - Scope tabs (All / Boards / Cards) rendered below the search input as `role="tab"` buttons.
  - Scope wired to the `type` query param on the search API via `useSearch`.
  - Search re-runs automatically on scope change (handled by `useSearch` dependency array).
  - When "All" scope is active, results are grouped into Boards / Cards sections.
  - When a specific scope is active, only the matching results section is shown.
  - Placeholder text is scope-aware: "Search boards and cards…" / "Search boards…" / "Search cards…".
  - Empty state messages are scope-aware: "No results found." / "No boards found." / "No cards found."
  - Scope is persisted to `sessionStorage` under the key `command-palette-scope`; survives palette close/reopen within the same browser session.
  - Keyboard navigation: ArrowDown from input focuses first result; ArrowUp from first result returns to input; ArrowRight/ArrowLeft navigate scope tabs with wrap-around; Tab from a scope tab moves focus to the first result when results are available.
  - Full dark-mode support via Tailwind `dark:` variants.
- `specs/tests/search-scope-tabs.md` — Playwright MCP test spec covering: tab presence, filtering per scope, scope persistence in sessionStorage, placeholder text per scope, scoped empty states, and keyboard navigation (tab arrows + result arrows).
