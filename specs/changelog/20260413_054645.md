## Technical Debt
- No dedicated `Badge` component exists; `BadgeSample` documents the canonical class pattern inline. A shared `Badge` component should be created in a future iteration once usage patterns stabilise.
- `ColorSwatch` resolves CSS variable values at paint time via `style={{ background: "var(--x)" }}`; resolved hex values are not shown as text. A future iteration could read `getComputedStyle` on mount to display the hex string.
