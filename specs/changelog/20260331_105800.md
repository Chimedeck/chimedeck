## New
- Added `terraform/modules/ecr/` module: defines an AWS ECR repository with scan-on-push, lifecycle policy (expire untagged images after 1 day, retain last 10 tagged), configurable mutability, and outputs for repository URL, ARN, and name.
- Added `terraform/modules/s3/` module: defines an S3 bucket with all public access blocked, server-side encryption (AES256), versioning enabled by default, and outputs for bucket name, ARN, and region.
- Both modules are parameterized and validated successfully with `terraform validate`.
