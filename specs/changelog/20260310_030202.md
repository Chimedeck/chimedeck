# Changelog — 2026-03-10T03:02:02Z

## Update
- `server/extensions/board/types.ts`: Added `BoardVisibility` type (`'PUBLIC' | 'PRIVATE' | 'WORKSPACE'`) and updated `Board` interface to include `visibility`, `description`, and `background` fields.

## New
- `db/migrations/0027_board_extensions.ts`: Migration adding `visibility` (enum, default `PRIVATE`), `description` (text, nullable), and `background` (varchar, nullable) columns to the `boards` table.
- `db/migrations/0028_card_start_date.ts`: Migration adding `start_date` (timestamp with tz, nullable) column to the `cards` table.
- `server/extensions/card/types.ts`: New shared `Card` interface including the new `start_date` field alongside all existing card columns.
