## New
- Added `terraform/modules/vpc/` module:
  - `main.tf`: Parameterized VPC, public/private subnets, IGW, and route tables.
  - `variables.tf`: Inputs for VPC CIDR, AZs, name prefix, tags, and create_vpc toggle.
  - `outputs.tf`: Outputs for VPC ID, subnet IDs, route table IDs, IGW ID.
- Added `terraform/modules/nat-gateway/` module:
  - `main.tf`: EIP, NAT Gateway, and private route table updates for NAT routing.
  - `variables.tf`: Inputs for name prefix, public subnet, private route tables, tags.
  - `outputs.tf`: Outputs for NAT Gateway ID and EIP public IP.
- Both modules pass `terraform validate`.
