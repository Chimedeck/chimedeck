## New
- `src/extensions/OfflineDrafts/storage.ts` — IndexedDB storage module using `idb`. Stores/retrieves/deletes `LocalDraft` objects keyed by `userId::workspaceId::cardId::draftType`. Provides `saveDraft`, `getDraft`, `getDraftsByCard`, `deleteDraft`. Degrades silently on IndexedDB failure.
- `src/extensions/OfflineDrafts/api.ts` — Typed client API wrapper for server draft endpoints (`listServerDrafts`, `upsertServerDraft`, `deleteServerDraft`). Attaches Bearer tokens and maps server error shapes to thrown rejections.
- `src/extensions/OfflineDrafts/reconcile.ts` — Last-write-wins reconciliation logic (`reconcileDrafts`, `reconcileDraftForType`). Compares `updatedAt` / `client_updated_at` timestamps; equal timestamps resolve to server to avoid pointless re-uploads.
