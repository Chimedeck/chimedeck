# Changelog — 20260302_130000

## Update
- `server/extensions/board/api/get.ts`: Now loads active lists from the `lists` table and returns them in `includes.lists` (previously a stub returning `[]`).
- `server/index.ts`: Mounted the `listRouter` under `/api/v1/boards/:id/lists` and `/api/v1/lists/:id`.

## New
- `db/migrations/0005_list.ts`: Migration adding the `lists` table (`id`, `board_id`, `title`, `position`, `archived`).
- `server/extensions/list/mods/fractional/initial.ts`: `firstPosition()` and sentinel constants for fractional indexing.
- `server/extensions/list/mods/fractional/index.ts`: `between(a, b)` for computing lexicographic midpoints; `generatePositions(n)` for fresh batch reorder.
- `server/extensions/list/api/create.ts`: `POST /api/v1/boards/:id/lists` — create list with fractional position; min role MEMBER.
- `server/extensions/list/api/list.ts`: `GET /api/v1/boards/:id/lists` — list active lists sorted by position; min role VIEWER.
- `server/extensions/list/api/update.ts`: `PATCH /api/v1/lists/:id` — rename list; min role MEMBER.
- `server/extensions/list/api/archive.ts`: `PATCH /api/v1/lists/:id/archive` — toggle archived; min role ADMIN.
- `server/extensions/list/api/delete.ts`: `DELETE /api/v1/lists/:id` — hard-delete (cascades via FK); min role ADMIN.
- `server/extensions/list/api/reorder.ts`: `POST /api/v1/boards/:id/lists/reorder` — batch reorder with count validation; min role MEMBER.
- `server/extensions/list/api/index.ts`: List router wiring all six endpoints.
- `src/extensions/List/api.ts`: Client API module for all list operations.
- `src/extensions/List/translations/en.json`: English copy for all list UI strings.
- `src/extensions/List/components/ListColumn.tsx`: Vertical card column component.
- `src/extensions/List/components/ListHeader.tsx`: List title bar with inline rename and archive/delete menu.
- `src/extensions/List/components/AddListButton.tsx`: Inline form to create a new list at end of board.
- `src/extensions/List/containers/BoardPage/ListColumn.tsx`: HTML5 drag-and-drop wrapper for `ListColumn`.
- `src/extensions/List/hooks/useListReorder.ts`: Optimistic reorder hook with server sync and rollback.
