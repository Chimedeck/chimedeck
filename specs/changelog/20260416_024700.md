# Sprint 137 — Iteration 5: CardModal.tsx banner + GIF file input support

## Update
- **CardModal.tsx — cover banner**: Replaced the fixed-height cover container (`h-44`/`h-28`) with an adaptive aspect-ratio container that uses `aspect-video` for 16:9 images and `aspect-square` for 1:1 or null aspect ratios. Color-only covers keep the fixed height strip.
- **CardModal.tsx — banner image**: Removed the `bg-slate-900/60` dark overlay class that was no longer needed; `object-contain` is retained so images are never cropped.
- **CardModal.tsx — file input**: Changed `accept` attribute from `image/*` to `image/jpeg,image/png,image/gif,image/webp`.
- **CardModal.tsx — validation**: Replaced the broad `startsWith('image/')` MIME guard with an explicit allowlist (`ALLOWED_COVER_TYPES`) and updated the error message to "Only JPEG, PNG, GIF, or WebP images can be used as a card cover."

_Nothing new beyond the above changes._
