# Changelog — 20260316_085632

## Update
- Added `ResizablePanels` component for the Sprint 81 card modal overhaul.

## New
- `src/extensions/Card/components/ResizablePanels.tsx` — two-column drag-to-resize layout:
  - Default ratio `0.55` / `0.45`
  - Persists ratio to `localStorage` under key `card_modal_column_ratio` on `mouseup` / `touchend`
  - Minimum column width `280px` enforced via clamping
  - Mouse and touch drag support
  - Keyboard support on the drag handle (`ArrowLeft` / `ArrowRight`, 1% step per keypress)
  - On mobile (`< 768px`) renders a vertical stack with no drag handle
- `specs/tests/card-resizable-panels.md` — Playwright test spec covering drag, persistence, default ratio, mobile layout, and keyboard accessibility.
