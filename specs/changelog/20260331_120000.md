# Changelog — 2026-03-31 12:00:00

## Update

- Added `terraform/modules/security-groups/` with four security groups: ALB (80/443 from internet), App (all TCP from ALB SG), DB (5432 from app SG), Redis (6379 from app SG). DB and Redis SGs have no egress rules.
- Added `terraform/modules/rds-postgres/` for PostgreSQL 16 with a custom parameter group enabling `pg_cron`, optional storage auto-scaling, optional Multi-AZ, `gp3` encrypted storage.

## New

- `terraform/modules/security-groups/main.tf` — four `aws_security_group` resources
- `terraform/modules/security-groups/variables.tf` — `name_prefix`, `vpc_id`, `tags`
- `terraform/modules/security-groups/outputs.tf` — `alb_sg_id`, `app_sg_id`, `db_sg_id`, `redis_sg_id`
- `terraform/modules/security-groups/README.md`
- `terraform/modules/rds-postgres/main.tf` — subnet group, pg16 parameter group with `pg_cron`, `aws_db_instance`
- `terraform/modules/rds-postgres/variables.tf` — all variables per sprint-125 spec
- `terraform/modules/rds-postgres/outputs.tf` — `endpoint`, `db_name`, `db_username`, `instance_id`
- `terraform/modules/rds-postgres/README.md` — includes sharding pattern documentation
