# Changelog — 2026-03-31 16:00:00

## Update
- `.umbra/production-fleet.yml`: New CI workflow template triggering on the `production-fleet` branch; sets `USE_SSH=TRUE`, `DEPLOYMENT_MODE=fleet`, passes all shared AWS secrets plus three fleet-specific secrets (`PRODUCTION_FLEET_ENCODED_PEM`, `PRODUCTION_FLEET_INSTANCE_URLS`, `PRODUCTION_FLEET_DEPLOY_SCRIPT`), and sequentially calls `scripts/build.sh` then `scripts/deploy.sh`.
- `.umbra/production-asg.yml`: New CI workflow template triggering on the `production-asg` branch; sets `USE_SSH=FALSE`, `DEPLOYMENT_MODE=asg-bluegreen`, `ENV_NAME=production`, `USE_RDS=TRUE`, passes shared AWS secrets plus two Terraform state secrets (`PRODUCTION_TF_STATE_BUCKET`, `PRODUCTION_TF_LOCK_TABLE`), and sequentially calls `scripts/build.sh` then `scripts/deploy.sh`.

## New
- `.umbra/production-fleet.yml` — CI pipeline for fixed-count fleet SSH deployments.
- `.umbra/production-asg.yml` — CI pipeline for ASG blue/green Terraform deployments.

## Technical Debt
- None introduced.
