# Sprint 18 — Board View (Kanban)

**Date:** 2026-03-02

---

## Update

- Replaced native HTML5 drag-and-drop in `BoardPage` with `@dnd-kit/core` + `@dnd-kit/sortable`.
- Refactored `BoardPage.tsx` to use the new `boardSlice` (replacing `BoardPage.duck.ts`) for full kanban state management.
- Updated `ListHeader.tsx` to dark kanban styling with Enter/Escape support for inline title editing.
- Updated `WorkspaceModal.test.tsx` and `WorkspaceShell.test.tsx` to include `board` reducer in mock stores (required after boardReducer was registered in the global store).

---

## New

- **`src/extensions/Board/slices/boardSlice.ts`**: Full kanban state (`board`, `listOrder`, `lists`, `cardsByList`, `cards`, `status`, `dragSnapshot`). Supports optimistic card move, list reorder, drag snapshot/rollback, and board title update actions.
- **`src/extensions/Board/api/board.ts`**: API wrapper re-exporting board endpoints.
- **`src/extensions/Board/api/list.ts`**: API wrapper re-exporting list endpoints.
- **`src/extensions/Board/api/card.ts`**: API wrapper re-exporting card endpoints; `moveCard` uses `PATCH /api/v1/cards/:id/move` per spec.
- **`src/extensions/Board/components/BoardHeader.tsx`**: Sticky board header with inline-editable title, connection indicator, `BoardMemberAvatars`, and settings menu (archive/delete).
- **`src/extensions/Board/components/BoardCanvas.tsx`**: `DndContext` wrapper with `DragOverlay`; horizontally scrollable `SortableContext` for lists; routes drag events to optimistic callbacks.
- **`src/extensions/Board/components/BoardMemberAvatars.tsx`**: Stacked avatar row showing up to 5 members with an overflow badge.
- **`src/extensions/Card/components/CardItem.tsx`**: Draggable card chip using `useSortable`; hides original during drag, shows `DragOverlay` with rotation/scale effect.
- **`src/extensions/Card/components/AddCardForm.tsx`**: Inline card creation form at column bottom; Enter submits, Escape dismisses.
- **`src/extensions/List/components/AddListForm.tsx`**: Inline list creation widget shown after the last column; styled with dashed border.
- **`src/extensions/List/containers/BoardPage/ListColumn.tsx`**: Replaced HTML5 drag with `useSortable` for list reorder; wraps `SortableContext` (vertical) for card ordering; renders `CardItem` + `AddCardForm`.
- `boardReducer` registered in `src/reducers.ts` and `src/store/index.ts`.
- Test specs added: `specs/tests/board-view.md`, `board-drag-card.md`, `board-add-card.md`.

## Technical Debt

- `BoardPage.duck.ts` still exists but is no longer used by `BoardPage.tsx`; it can be removed in a future cleanup pass.
- `BoardPage.tsx` references `__api__` via `globalThis` instead of the Redux extra argument — matches existing patterns but should be unified.
- Pre-existing TypeScript errors in `BoardListPage.duck.ts` and `BoardPage.duck.ts` (`extra` typed as `unknown`) are not fixed; they predate sprint 18.
- `BoardMemberAvatars` receives no real member data yet — the board API would need to include a `members` include for this to populate.
