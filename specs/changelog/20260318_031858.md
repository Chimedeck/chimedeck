## New
- `server/extensions/search/mods/queryWorkspaceSearch.ts` — access-aware workspace search mod that enforces board-level visibility rules at the SQL layer. Filters boards/cards by caller role: OWNER/ADMIN see all boards; MEMBER/VIEWER see PUBLIC + WORKSPACE + explicit PRIVATE; GUEST sees PUBLIC + explicitly granted PRIVATE only (WORKSPACE excluded).
