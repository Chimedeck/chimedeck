## New
- **`src/layout/hooks/useSidebarState.ts`**: Custom hook that reads/writes `sidebar_collapsed` to `localStorage` for state persistence. Returns `{ collapsed, toggle }`.
- **`src/layout/Sidebar.tsx`**: Layout-level collapsible sidebar. Expands to `w-64` (labels + icons) and collapses to `w-16` (icon-only rail with CSS tooltips). Includes toggle button with `ChevronLeft`/`ChevronRight` icons and `data-testid="sidebar-toggle"` / `data-testid="sidebar"` / `data-collapsed` attributes for test targeting. Workspace switcher, nav links, and user menu each have collapsed variants with accessible `aria-label` attributes and hover tooltips.
- **`specs/tests/sidebar-collapsible-drawer.md`**: Playwright MCP test scenarios covering desktop toggle, icon/label visibility, tooltip behavior, keyboard accessibility, localStorage persistence, and mobile layout safety.
