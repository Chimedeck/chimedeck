## New

  - adds a partial unique index on `(user_id, source_type, source_id, type)` for `source_type = 'board_activity'`.
- Added `createActivityEvent` regression tests to lock notification fan-out behavior for:
  - `emitCardCreated`
  - `emitCardMoved`
  - same-list move no-op

- Extended `boardActivityDispatch` test coverage with recipient deduplication when a user is both board member and board guest.
