# Changelog — 20260302_045000

## Update
- `server/index.ts`: Mounted the board router under `/api/v1/workspaces/:id/boards` and `/api/v1/boards`.

## New
- `db/migrations/0004_board.ts`: Migration creating the `boards` table (`id`, `workspace_id`, `title`, `state ACTIVE|ARCHIVED`, `created_at`) with FK to workspaces.
- `server/extensions/board/middlewares/requireBoardWritable.ts`: Middleware that loads a board by ID and returns 403 if it is ARCHIVED; attaches the board to the request on success.
- `server/extensions/board/api/create.ts`: `POST /api/v1/workspaces/:id/boards` — creates a board with `state: ACTIVE`; min role MEMBER.
- `server/extensions/board/api/list.ts`: `GET /api/v1/workspaces/:id/boards` — lists all boards (active + archived); min role VIEWER.
- `server/extensions/board/api/get.ts`: `GET /api/v1/boards/:id` — returns board with shallow `includes: { lists, cards }` (empty arrays until sprint 06/07); min role VIEWER.
- `server/extensions/board/api/update.ts`: `PATCH /api/v1/boards/:id` — renames board; requires `requireBoardWritable` + ADMIN role.
- `server/extensions/board/api/archive.ts`: `PATCH /api/v1/boards/:id/archive` — toggles ACTIVE ↔ ARCHIVED; min role ADMIN.
- `server/extensions/board/api/delete.ts`: `DELETE /api/v1/boards/:id` — hard-deletes board; min role ADMIN.
- `server/extensions/board/api/duplicate.ts`: `POST /api/v1/boards/:id/duplicate` — deep-copies board via `duplicateBoard` mod; min role MEMBER.
- `server/extensions/board/api/index.ts`: Board API router — pattern-matches all 7 board routes and delegates to handlers.
- `server/extensions/board/mods/duplicate/index.ts`: Transactional deep-copy mod — copies board → lists → cards (lists/cards stubbed until sprint 06/07).
- `src/extensions/Board/api.ts`: Client API for all board endpoints (listBoards, getBoard, createBoard, updateBoard, archiveBoard, deleteBoard, duplicateBoard).
- `src/extensions/Board/components/BoardCard.tsx`: Board summary tile with archive/delete/duplicate actions.
- `src/extensions/Board/components/BoardStateChip.tsx`: ACTIVE/ARCHIVED badge component.
- `src/extensions/Board/components/CreateBoardModal.tsx`: Modal for creating a new board with title input.
- `src/extensions/Board/containers/BoardListPage/BoardListPage.tsx`: Board list page — fetches and renders all boards with create/archive/delete/duplicate controls.
- `src/extensions/Board/containers/BoardListPage/BoardListPage.duck.ts`: Redux duck for board list state — thunks and reducers for all board mutations.
- `src/extensions/Board/containers/BoardPage/BoardPage.tsx`: Single board page showing title, archived notice, and placeholder for lists/cards.
- `src/extensions/Board/containers/BoardPage/BoardPage.duck.ts`: Redux duck for single board state.
- `src/extensions/Board/routes.ts`: Board routes (`/workspace/:workspaceId/boards`, `/boards/:boardId`).
- `src/extensions/Board/translations/en.json`: English translations for board UI strings.
- `specs/tests/board.test.ts`: 12 unit tests covering middleware logic, duplicate title, state transitions, input validation, and list/delete behaviour.
