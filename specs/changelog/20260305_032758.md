# Changelog — 20260305_032758

## Update
- `server/extensions/plugins/api/registry/list.ts`: Extended `handleListPlugins` to support query params `q` (ILIKE search on name/description/author), `category` (JSONB array contains), `isPublic` (boolean filter), `page` and `perPage` (pagination, max 50, default 20). Soft-deleted plugins (`is_active=false`) are now always excluded. Response now includes `metadata: { total, totalPage, page, perPage }`.
- `server/extensions/plugins/api/index.ts`: Added import and route for `GET /api/v1/plugins/categories`, registered before `/:pluginId` to avoid route collision.

## New
- `server/extensions/plugins/api/registry/categories.ts`: New handler for `GET /api/v1/plugins/categories` — returns a deduplicated, sorted array of category strings extracted from all active plugins via `jsonb_array_elements_text`.

## Technical Debt
- None introduced.
