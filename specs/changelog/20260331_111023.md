## New
- Added `terraform/modules/elasticache-redis/` module with the following files:
  - `main.tf`: Defines ElastiCache Redis 7 resources supporting both single-node and cluster mode, with parameterized configuration for engine version, node type, node count, and cluster mode toggle.
  - `variables.tf`: Declares all input variables, including cluster mode, node type, subnet IDs, security group, encryption options, and tags.
  - `outputs.tf`: Exposes primary endpoint, reader endpoint (for cluster mode), and cluster ID.
  - `README.md`: Documents module usage, variables, outputs, and provides example configurations for both single-node and cluster mode.
