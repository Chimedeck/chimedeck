## New
- `server/extensions/plugins/api/board-plugins/available.ts` — `GET /api/v1/boards/:boardId/plugins/available` endpoint. Returns globally active plugins (`is_active = true`) that are NOT currently enabled on the given board (i.e., no `board_plugins` row with `disabled_at IS NULL`). Auth: board member via `boardMemberGuard`. Supports `q` (ILIKE on name/description/author) and `category` (JSONB contains) query params. Response: `{ data: Plugin[], metadata: { total: number } }`.
