# Changelog — 2026-03-23 07:58

## Update
- `package.json`: added `@modelcontextprotocol/sdk@1.27.1` dependency (installed via `bun add`).

## New
- `server/extensions/mcp/config.ts`: reads `HORIFLOW_TOKEN` and `HORIFLOW_API_URL` from environment. Exits with a clear, actionable error message (exit code 1) if `HORIFLOW_TOKEN` is missing.
- `server/extensions/mcp/apiClient.ts`: thin fetch wrapper that injects `Authorization: Bearer <token>` on every request and normalises error responses to `{ error: { name, data? } }`.
- `server/extensions/mcp/index.ts`: MCP server entry point (shebang `#!/usr/bin/env bun`). Instantiates `McpServer` with `StdioServerTransport`; no tools registered yet (deferred to subsequent iterations).
