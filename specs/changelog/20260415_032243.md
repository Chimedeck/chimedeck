- `src/extensions/Webhooks/containers/WebhooksRegisterPage/RegisterWebhookModal.tsx`: Modal for registering a new webhook endpoint. Features: label (text, max 100 chars), endpoint URL (https:// validation), grouped event-type checklist (6 groups per spec), select-all / clear-all per group, and a guard that prevents submission if no event types are selected.
- `src/extensions/Webhooks/containers/WebhooksRegisterPage/WebhookCreatedModal.tsx`: One-time signing secret reveal modal shown after successful registration. Includes a success banner, monospace secret display, copy-to-clipboard button (resets to "Copy" after 2 s), and a warning callout that the secret will not be shown again.
- `src/extensions/Webhooks/containers/WebhooksRegisterPage/index.ts`: Re-exports `WebhooksRegisterPage`, `RegisterWebhookModal`, and `WebhookCreatedModal` for clean consumer imports.
- `tests/unit/webhooks/RegisterWebhookModal.test.tsx`: 18 unit tests covering URL validation (https accepted, http/ftp/empty rejected), event-type validation (empty set fails), select-all / clear-all set operations, and event group structure correctness.
- `tests/unit/webhooks/WebhookCreatedModal.test.tsx`: 9 unit tests covering copy-to-clipboard state machine (immediate true, resets after timeout, does not reset before timeout), signing secret shape, and all translation key presence.
