## New
- Added `terraform/modules/ec2-single/` module:
  - `main.tf`: Provisions a single EC2 instance with EIP, user data via `templatefile`, and lifecycle settings.
  - `variables.tf`: Declares all required input variables (AMI, instance type, subnet, SGs, user data, tags, etc.).
  - `outputs.tf`: Outputs instance ID, public IP, EIP allocation ID, and private IP.
  - `README.md`: Usage, variables, outputs, and notes.
- Added `terraform/modules/vpc-peering/` module:
  - `main.tf`: Cross-region VPC peering with provider aliases, bidirectional routes, and required provider block.
  - `variables.tf`: Declares all required input variables (VPC IDs, regions, CIDRs, route tables).
  - `outputs.tf`: Outputs peering connection ID and status.
  - `README.md`: Usage with provider aliases, variables, outputs, and notes.
