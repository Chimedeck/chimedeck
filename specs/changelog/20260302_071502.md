## Technical Debt

- Search results from boards and cards are fetched in two separate queries and merged in memory; a single UNION query would be more efficient at scale.
- Cursor pagination uses `id > cursor` which is only correct if IDs are monotonically sortable (UUIDs are not strictly ordered).
