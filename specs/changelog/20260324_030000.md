# Changelog — 2026-03-24

## Update
- `Makefile`: added `test-e2e` and `test-all` targets alongside the existing `test` target. Updated `.PHONY` declaration accordingly.

## New
- `run-all-tests-full.sh`: orchestration script that runs all three test modes in sequence:
  1. **Unit/integration** — `bun test tests/integration`
  2. **Playwright e2e** — `npx playwright test --project=e2e`
  3. **MCP scenarios** — `bash run-all-tests.sh`

  Supports `SKIP_UNIT`, `SKIP_E2E`, `SKIP_MCP` env vars to skip individual modes. Forwards `BASE_URL` as `TEST_BASE_URL` to Playwright. Soft-fails with a warning when a required tool (`bun`, `npx`) or directory is missing. Exits non-zero when any enabled mode fails. Prints a colour-emoji summary table at the end.
