## Update
- Updated mention resolution to respect board visibility rules:
  - `PRIVATE` boards keep existing behavior (explicit board participants only).
  - `WORKSPACE` and `PUBLIC` boards now allow mentions for any non-guest workspace member, even when not explicitly added to `board_members`.
- Updated mention autocomplete suggestions to match the same visibility-aware scope logic used by mention persistence.
- Updated mention autocomplete suggestions to support searching by email while keeping the displayed suggestion label as the user name.
- Fixed realtime board card comment counters so comment badges update outside the card modal when other users add comments.
- Fixed card-modal comment mutations to use the board card as the counter source of truth (instead of card-detail payload), preventing incorrect resets like `3 -> 1` after posting a comment.

## New
- Visibility-aware mention scope behavior for:
  - `server/common/mentions/resolve.ts`
  - `server/extensions/board/api/members/suggestions.ts`

## Technical Debt
