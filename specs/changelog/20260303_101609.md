# Changelog — 20260303_101609

## Update

- **server/extensions/card/api/list.ts** — Extended the SQL query with a `LEFT JOIN` on `card_labels` and `labels` tables; aggregates label data as a JSON array (`id`, `name`, `color`) per card using `json_group_array`. Null labels (cards with no labels) are filtered out. Each card in the response now includes a `labels` field.
- **src/extensions/Card/api.ts** — Added `labels?: Array<{ id: string; name: string; color: string }>` to the `Card` interface so the client type matches the updated server response.
- **src/extensions/Card/components/CardMemberAvatars.tsx** — Added optional `currentUserId?: string` prop (ignored for now; Sprint 28 will wire up the member popover logic).
- **src/extensions/Card/components/CardItem.tsx** — Added `labelsExpanded` and `onToggleLabels` props; renders `<CardLabelChips>` above the card title when `card.labels` is non-empty; chip clicks call `onToggle` with `e.stopPropagation()` to avoid opening the card modal; also renders `<CardMemberAvatars>` when members are present (Sprint 28 stub).
- **src/extensions/List/containers/BoardPage/ListColumn.tsx** — Added `labelsExpanded` and `onToggleLabels` props; passes them (with `?? false` / `?? (() => {})` defaults) to every `<CardItem>`.
- **src/extensions/Board/components/BoardCanvas.tsx** — Added `boardId: string` prop; calls `useCardLabelExpanded(boardId)` to get `[labelsExpanded, onToggleLabels]`; passes both to `<SortableListColumn>` and the DragOverlay `<CardItem>`.
- **src/extensions/Board/containers/BoardPage/BoardPage.tsx** — Passes `boardId={boardId ?? ''}` to `<BoardCanvas>`.
- **src/extensions/Board/slices/boardSlice.ts** — Added `as Card` cast in `remoteCardMove` spread to satisfy `exactOptionalPropertyTypes: true` with the new optional `labels` field.

## New

- **src/extensions/Card/hooks/useCardLabelExpanded.ts** — New React hook that persists the label chip expanded/collapsed toggle state per board in `localStorage` under the key `card-labels-expanded:<boardId>`. Defaults to `false`. Returns `[expanded: boolean, toggle: () => void]`.
- **src/extensions/Card/components/CardLabelChips.tsx** — New presentational component. Renders coloured label pills on a card tile. Collapsed state: short `h-2` coloured bars with `title` tooltip. Expanded state: pill badges with label name text. Supports `aria-expanded` and `role="button"` for accessibility. Clicks are stop-propagated so they do not open the card modal.
