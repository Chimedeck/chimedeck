# Changelog — 2026-03-10T04:24:23Z

## Update

- Replaced all legacy `{ name: "code", data: { message: "…" } }` API error responses across 102 server-side files (92 in `server/extensions/`, 6 in `server/middlewares/`, and `server/index.ts`) with the standard error envelope: `{ error: { code: "…", message: "…" } }`.
