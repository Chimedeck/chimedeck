# Changelog — 2026-03-31T03:49:55Z

## Update
- `terraform/modules/vpc/` — added full VPC module with conditional creation (`create_vpc` bool), public/private subnets (one per AZ), IGW, public route table, and per-AZ private route tables ready for NAT GW wiring.
- `terraform/modules/nat-gateway/` — added NAT Gateway module that allocates an EIP, creates the NAT GW in a specified public subnet, and adds `0.0.0.0/0 → nat-gateway` routes to all supplied private route tables.

- No secondary CIDR support — extend if VPC CIDR expansion is ever needed.
