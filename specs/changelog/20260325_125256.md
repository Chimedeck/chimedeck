## Technical Debt
- The `text-base` class name serves dual purpose: Tailwind's built-in font-size utility (`1rem`) and the project's semantic text-colour token (`var(--text-base)`). Both CSS properties apply when the class is used, which is acceptable for body text but could cause unintended font-size resets in edge cases.
- Pre-existing TypeScript `exactOptionalPropertyTypes` errors (8 errors) on `NavItem`/`NavButton` component usages in `Sidebar.tsx` remain unaddressed — pre-existing before this iteration.
