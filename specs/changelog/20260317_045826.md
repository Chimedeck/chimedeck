# Changelog — 2026-03-17T04:58:26Z

## Update
- `OneLineToolbar.tsx`: Replaced inline overflow dropdown with shared `CommandMenu` component. Removed now-unused icon imports (`NumberedListIcon`, `ChatBubbleLeftEllipsisIcon`, `CodeBracketIcon`).

## New
- `src/extensions/Card/components/CommandMenu.tsx`: New shared component for the + overflow command menu in rich text editors. Features:
  - Search input auto-focused on open; filters command list by label/keywords (case-insensitive)
  - Five commands: Mention (inserts `@`), Emoji (inserts `:`), Code snippet, Quote, Numbered list
  - Empty-state message "No matching commands" when query yields no results
  - Keyboard navigation: ArrowUp/ArrowDown to move, Enter to execute, Escape to close
  - Active formatting state (e.g. cursor inside blockquote) visually highlighted
  - `onMouseDown` with `preventDefault` prevents editor blur during selection
- `specs/tests/rich-text-toolbar-single-line-overflow.md`: Added three Playwright MCP scenario sections for + menu search/filter in both editors and for Mention/Emoji commands.
