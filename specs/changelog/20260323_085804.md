# Changelog — 2026-03-23T08:58:04Z

## Update
- Fixed `horiflow <command> --help` routing: global `--help` no longer intercepts subcommand help flags. Command-specific help is now shown correctly (e.g., `horiflow move-card --help`).

## New
- `cli/commands/createCard.ts` — Implements `create-card` command (`POST /api/v1/lists/:listId/cards`). Flags: `--list` (required), `--title` (required), `--description` (optional). Prints created card ID on success; supports `--json` output.
- `cli/commands/editDescription.ts` — Implements `edit-description` command (`PATCH /api/v1/cards/:cardId/description`). Flags: `--card` (required), `--description` (required). Prints confirmation on success; supports `--json` output.
- `cli/index.ts` — Wired `create-card` and `edit-description` into the command dispatcher.
