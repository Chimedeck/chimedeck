- `server/extensions/board/api/follow.ts`: Handlers for `POST /api/v1/boards/:id/follow` (idempotent) and `DELETE /api/v1/boards/:id/follow` (idempotent).
- `server/extensions/board/api/me-starred-boards.ts`: Handler for `GET /api/v1/me/starred-boards` — returns all boards starred by the current user.
- `tests/integration/board/starFollow.test.ts`: Integration tests verifying 401 auth guards for all 5 new endpoints.
