# Changelog — 2026-03-23T16:00:00Z

## Update
- `cli/index.ts`: Wired `set-price` and `invite` commands into the CLI dispatcher, imported new command modules.

## New
- `cli/commands/setPrice.ts`: Implements the `set-price` subcommand — PATCH `/api/v1/cards/:cardId/money`. Flags: `--card` (required), `--amount`, `--currency`, `--label`, `--clear`. `--clear` takes precedence over `--amount`/`--currency` and sends `{ amount: null }`. Validates required flags, supports `--help` and `--json`.
- `cli/commands/invite.ts`: Implements the `invite` subcommand — POST `/api/v1/boards/:boardId/members`. Flags: `--board` (required), `--email` (required), `--role` (default: `member`). Validates required flags, surfaces 403 errors clearly via the shared apiClient error handler, supports `--help` and `--json`.
