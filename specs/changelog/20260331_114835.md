## New
- Added `terraform/scripts/startup.sh`: EC2 user data template for Docker-based app startup. Handles Docker enablement, ECR login, image pull, secrets fetch from AWS Secrets Manager, `.env` file creation, and app launch via Docker Compose. Uses Terraform template variables for ECR image URL and secrets ARN, with correct escaping for bash and Terraform interpolation.
- Added `terraform/scripts/wait-for-healthy-targets.sh`: Polling script for AWS ELBv2 target group health. Accepts target group ARN, minimum healthy count, and timeout. Exits 0 when healthy targets threshold is met, 1 on timeout or AWS CLI error. Includes argument validation and robust error handling.
