# Changelog — 20260312_034128

## Update

- `src/extensions/Automation/components/AutomationPanel/index.tsx`: Activated the **Schedule** tab — replaced the "Coming soon" placeholder with `SchedulePanel`.

## New

- `src/extensions/Automation/components/SchedulePanel/index.tsx`: Root SchedulePanel component. Renders quick-start templates, a divider, and either `ScheduleList` (if automations exist) or `ScheduleEmptyState`. Manages builder mode state (null / scheduled / duedate) and renders the corresponding builder in-place.

- `src/extensions/Automation/components/SchedulePanel/ScheduleList.tsx`: Lists SCHEDULED and DUE_DATE automations in two labelled sections, each with a "+ Add" button. Delegates rows to `ScheduleItem`.

- `src/extensions/Automation/components/SchedulePanel/ScheduleItem.tsx`: Row component for a single scheduled/due-date automation. Shows a type-appropriate icon (ClockIcon / ExclamationCircleIcon), name, schedule summary (from `scheduleSummary` / `dueDateSummary`), action count. Hover-reveal: enable/disable toggle, edit, two-click delete.

- `src/extensions/Automation/components/SchedulePanel/ScheduleEmptyState.tsx`: Empty state shown when no SCHEDULED or DUE_DATE automations exist. Provides two CTA buttons.

- `src/extensions/Automation/components/SchedulePanel/QuickStartTemplates.tsx`: Three quick-start template cards (Weekly board cleanup, Overdue flagging, Monthly archive). Clicking a card calls `onUseTemplate` with a `QuickStartTemplate` object containing pre-populated trigger/action config, opening the appropriate builder.

- `tests/e2e/automation/scheduledCommands.md`: Playwright MCP tests for scheduled command creation, quick-start templates, enable/disable, delete, empty state, and summary text.

- `tests/e2e/automation/dueDateCommands.md`: Playwright MCP tests for due date command creation (before/on/after), summary text accuracy, edit, delete, toggle, template usage, and section isolation.
