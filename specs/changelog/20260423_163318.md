## Update
- Improved cross-column drag insertion accuracy in [src/extensions/Board/components/BoardCanvas.tsx](src/extensions/Board/components/BoardCanvas.tsx) by using pointer-only viewport midpoint comparisons for destination-list index detection.
- Improved same-list pointer-index stability in long columns by guarding null-adjusted pointer values during scroll-aware midpoint resolution.
- Hardened drag-end destination reconciliation so pointer-fallback recomputation only overrides placeholder-derived destination when there is a real mismatch.
- Fixed cross-column index pinning where stale `over.id` could lock insertion to the first recognized destination card while dragging downward; pointer-derived index is now primary and card-hover override is only applied when locally consistent.
- Removed placeholder-height midpoint compensation from cross-column detection because it could over-correct insertion thresholds and misplace hover index.
- Fixed cross-column hover lag where destination index stopped updating after first column entry: cross-list placeholder index is now recomputed in the pointermove rAF loop (not only `onDragOver`), which avoids reliance on sparse over-events from single-container destination columns.
- Reduced drag-frame React render overhead in list columns by stabilizing all `ListHeader` action callbacks in [src/extensions/List/containers/BoardPage/ListColumn.tsx](src/extensions/List/containers/BoardPage/ListColumn.tsx) and memoizing [src/extensions/List/components/ListHeader.tsx](src/extensions/List/components/ListHeader.tsx) with a focused prop comparator.
