# Iteration 10: Plugin Popup + Modal + show-settings + End-to-End Wiring

## Update
- Enhanced `usePluginBridge.ts` to handle all plugin UI message types (modal/popup open, close, update, size-to) with new callback props.
- Updated `PluginIframeContainer.tsx` to manage modal/popup state and render overlays.
- Modified `PluginDashboardPage.tsx` to support a settings gear icon that opens the plugin settings modal.
- Updated `PluginList.tsx` and `PluginCard.tsx` to support and display the settings gear for eligible plugins.

## New

- Added `src/extensions/Plugins/modals/PluginPopup.tsx`: Floating popup overlay for plugin iframes, positioned contextually.
- Implemented Redux/modal state and UI flows for plugin modal and popup management.
