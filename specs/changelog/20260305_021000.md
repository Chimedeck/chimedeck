## New
- `server/extensions/plugins/api/board-plugins/list.ts` — GET handler that returns active board plugins (disabled_at IS NULL) joined with plugin metadata.
- `server/extensions/plugins/api/board-plugins/enable.ts` — POST handler (board admin only) that enables a plugin; re-enables previously disabled rows; returns `plugin-not-active` or `plugin-already-enabled` error names for edge cases.
- `server/extensions/plugins/api/board-plugins/disable.ts` — DELETE handler (board admin only) that soft-disables a plugin by setting `disabled_at`; row is not deleted.
