## New
- `src/extensions/Board/containers/BoardSettings/BoardNotificationToggle.tsx` — per-board notification toggle for the current user. On mount it GETs `/api/boards/:boardId/notification-preference`. On toggle it PATCHes the same endpoint with optimistic update and rollback on failure. Defaults to enabled (opt-out model) when the API call fails.
