# Sprint 03 ‚Äî Authentication

## Update

Implemented full authentication system: email/password login, JWT (RS256) access tokens, rotating httpOnly refresh tokens, OAuth (Google + GitHub), and user profile endpoints.

## New

- **`db/migrations/0002_auth.ts`** ‚Äî Knex migration creating `users` and `refresh_tokens` tables
- **`server/common/db.ts`** ‚Äî shared Knex instance reused by all extensions
- **`server/mods/cache/index.ts`** ‚Äî in-memory TTL cache (node-cache) for OAuth state nonces and rate-limit counters
- **`server/config/env.ts`** ‚Äî extended with `APP_URL`, `OAUTH_GOOGLE_CLIENT_ID/SECRET`, `OAUTH_GITHUB_CLIENT_ID/SECRET`
- **`server/extensions/auth/`** ‚Äî complete auth extension:
  - `common/config/jwt.ts` ‚Äî RS256 key config (15-min access token, 7-day refresh token)
  - `common/config/oauth.ts` ‚Äî Google + GitHub OAuth provider config
  - `mods/token/issue.ts` ‚Äî issue RS256 JWT access tokens (via `jose`)
  - `mods/token/verify.ts` ‚Äî verify + decode access tokens
  - `mods/token/refresh.ts` ‚Äî rotate refresh tokens (revoke old, insert new)
  - `mods/password/hash.ts` ‚Äî bcrypt hash via `Bun.password` (cost 12)
  - `mods/password/verify.ts` ‚Äî bcrypt compare
  - `mods/oauth/google.ts` ‚Äî exchange Google auth code for user profile
  - `mods/oauth/github.ts` ‚Äî exchange GitHub auth code for user profile (with email fallback)
  - `middlewares/authentication.ts` ‚Äî Bearer token middleware (deny-first)
  - `api/login.ts` ‚Äî `POST /api/v1/auth/token` with IP-based rate limiting (10 req/min)
  - `api/refresh.ts` ‚Äî `POST /api/v1/auth/refresh` (reads httpOnly cookie)
  - `api/logout.ts` ‚Äî `DELETE /api/v1/auth/session` (revokes refresh token, clears cookie)
  - `api/oauth/index.ts` ‚Äî `GET /api/v1/auth/oauth/:provider` (redirect with CSRF nonce)
  - `api/oauth/callback.ts` ‚Äî `GET /api/v1/auth/oauth/:provider/callback` (upsert user, issue tokens)
  - `api/index.ts` ‚Äî auth router
- **`server/extensions/users/`** ‚Äî users extension:
  - `api/me.ts` ‚Äî `GET /api/v1/users/me` + `PATCH /api/v1/users/me`
  - `api/index.ts` ‚Äî users router
- **`server/index.ts`** ‚Äî updated to mount auth and users routers
- **Tests**: `token.test.ts` (issue, verify, tamper) + `password.test.ts` (hash, verify, reject) ‚Äî 6 new tests, all passing
- **Packages added**: `jose@6.1.3`, `uuid@13.0.0`, `@types/uuid@11.0.0`
