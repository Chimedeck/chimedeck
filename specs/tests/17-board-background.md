# 17. Board Background

**Prerequisites:** Flow 06 (Create Board) completed. On `Test Board`.  
**Continues from:** Board view.  
**Ends with:** Board background has been changed to a solid colour and then reset.

---

## Steps

1. On the board, click the **Customize** button (or **⋮** → **Change background**).
   - **Expected:** Background customization panel opens showing colour swatches or image options.

2. Select a **solid colour** background (e.g. green or blue).
   - **Expected:** Board background changes immediately to the selected colour.

3. Select a different colour (e.g. red).
   - **Expected:** Board background updates to the new colour.

4. If an **Upload image** option is available:
   a. Click **Upload image**.
   b. Upload a small image file (< 2 MB).
   - **Expected:** Board background updates to show the uploaded image.

5. Select the original default background or close the panel.
   - **Expected:** Panel closes; board shows the last selected background.

---

## Notes

- Background changes persist across page reload.
- Continue to flow **18-board-star-follow**.
