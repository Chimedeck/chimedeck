# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in ChimeDeck, please report it responsibly by emailing [developer@journeyh.io](mailto:developer@journeyh.io) rather than opening a public issue.

You can expect an initial response within 48 hours. We will keep you informed of progress as we investigate and, if confirmed, work to release a fix as quickly as possible. If the vulnerability is accepted, we will credit you in the release notes (unless you prefer to remain anonymous). If it is declined, we will explain why.

Please do not disclose the vulnerability publicly until a fix has been released.
